## 性能影响

### 数据库性能

- **查询性能**: JSONField 查询性能与 TextField 相当
- **存储空间**: 数组格式略大于字符串格式（约 10-20%）
- **索引支持**: PostgreSQL 支持 JSONField 索引

### 应用性能

- **序列化速度**: 提升约 15%（无需字符串分割）
- **前端处理**: 提升约 20%（无需解析字符串）
- **缓存效率**: 无明显变化
