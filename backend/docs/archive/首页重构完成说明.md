# 首页重构完成说明

## 完成时间
2026-03-02

## 重构目标
将管理后台首页从"管理向"改为"用户向",展示对普通用户有价值的统计信息和内容推荐。

## 已完成的工作

### 1. 后端API实现 ✅

#### 文件位置
- `backend/mainotebook/system/views/home_stats.py`
- `backend/mainotebook/system/urls.py`

#### API端点
所有端点基础路径: `/api/system/home_stats/`

1. **平台统计** - `GET /platform_stats/`
   - 知识库总数
   - 人设卡总数
   - 用户总数
   - 总下载次数
   - 权限: 所有人可访问

2. **个人统计** - `GET /my_stats/`
   - 我的上传数
   - 我的下载数
   - 我的评论数
   - 我的待审核数
   - 权限: 需要登录

3. **热门内容** - `GET /hot_content/`
   - 热门知识库列表(前5)
   - 热门人设卡列表(前5)
   - 按下载量排序
   - 权限: 所有人可访问

4. **最新内容** - `GET /recent_content/`
   - 最新知识库列表(前5)
   - 最新人设卡列表(前5)
   - 权限: 所有人可访问

5. **月度趋势** - `GET /monthly_trend/`
   - 最近12个月的数据
   - 知识库上传数量
   - 人设卡上传数量
   - 权限: 所有人可访问

### 2. 前端实现 ✅

#### 文件位置
- `admin_web/src/views/system/home/api.ts` (新建)
- `admin_web/src/views/system/home/index.vue` (修改)

#### 主要修改

1. **顶部统计卡片** (4个)
   - 知识库总数 (蓝色图标)
   - 人设卡总数 (绿色图标)
   - 我的上传 (橙色图标)
   - 我的下载 (红色图标)
   - 移除了百分比显示

2. **折线图 - 内容上传趋势**
   - 标题: "内容上传趋势"
   - 显示最近12个月的知识库和人设卡上传数量
   - 两条曲线: 知识库(橙色)、人设卡(紫色)
   - 带渐变填充效果

3. **柱状图 - 热门内容下载排行**
   - 标题: "热门内容下载排行"
   - 显示下载量前10的内容
   - 横向柱状图
   - 知识库(橙色)、人设卡(紫色)区分显示

4. **快捷导航**
   - 浏览知识库
   - 浏览人设卡
   - 我的上传
   - 我的下载
   - 消息中心
   - 个人设置

5. **移除的内容**
   - 删除了饼图组件
   - 移除了管理向的快捷导航(角色、部门、配置等)
   - 移除了订单统计、月度计划、访问统计等管理数据

### 3. 数据加载逻辑 ✅

#### 新增函数
- `loadStats()`: 加载平台统计和个人统计数据
- 在 `onMounted` 生命周期中调用
- 错误处理: 失败时显示默认值,不影响页面渲染

#### API调用
- 使用 `homeApi` 模块调用后端接口
- 异步加载,不阻塞页面渲染
- 支持深色主题切换时重新渲染图表

## 技术特点

### 后端
- 使用 Django REST Framework 的 ViewSet
- 权限控制: AllowAny 和 IsAuthenticated
- 数据聚合: 使用 Django ORM 的 Count 和 annotate
- 时间处理: 使用 timezone 和 timedelta 计算月度数据

### 前端
- Vue 3 Composition API
- ECharts 图表库
- 响应式设计
- 支持深色主题
- 错误处理和降级显示
