# 知识库广场页面配置说明

## 页面信息
- 页面路径: `/content/knowledge`
- 组件路径: `admin_web/src/views/content/knowledge/index.vue`
- API路径: `admin_web/src/views/content/knowledge/api.ts`

## 后端路由配置

由于项目使用后端控制路由,需要在数据库的菜单表中添加以下配置:

### 菜单配置

```json
{
  "name": "知识库广场",
  "path": "/content/knowledge",
  "component": "content/knowledge/index",
  "meta": {
    "title": "知识库广场",
    "icon": "fa fa-book",
    "isLink": "",
    "isHide": false,
    "isKeepAlive": true,
    "isAffix": false,
    "isIframe": false,
    "roles": ["admin", "user"],
    "auth": []
  }
}
```

### 配置步骤

1. 登录管理后台
2. 进入"系统管理" -> "菜单管理"
3. 添加新菜单,填写以下信息:
   - 菜单名称: 知识库广场
   - 路由地址: /content/knowledge
   - 组件路径: content/knowledge/index
   - 菜单图标: fa fa-book
   - 是否缓存: 是
   - 是否隐藏: 否

4. 保存后,刷新页面即可在侧边栏看到新菜单

## 前端快捷导航配置

## API端点

知识库广场使用以下API端点:

- `GET /api/content/knowledge/` - 获取知识库列表
- `GET /api/content/knowledge/{id}/` - 获取知识库详情
- `POST /api/content/knowledge/{id}/star/` - 收藏知识库
- `DELETE /api/content/knowledge/{id}/star/` - 取消收藏
- `GET /api/content/knowledge/{id}/files/{file_id}/` - 下载文件

## 功能特性

1. **分类筛选**
   - 全部
   - 热门推荐
   - 最新上传
   - 我的收藏

2. **标签筛选**
   - 支持多标签筛选
   - 热门标签快速选择

3. **搜索功能**
   - 关键词搜索
   - 实时搜索

4. **卡片展示**
   - 网格布局
   - 悬停动画
   - 显示下载数和收藏数

5. **详情弹窗**
   - 基本信息展示
   - 文件列表
   - 收藏/取消收藏
   - 文件下载

6. **分页功能**
   - 支持切换每页数量
   - 页码跳转

## 注意事项

1. 确保后端API已正确配置并可访问
2. 确保用户有访问权限
3. 文件下载功能需要正确的文件存储配置
4. 收藏功能需要用户登录
