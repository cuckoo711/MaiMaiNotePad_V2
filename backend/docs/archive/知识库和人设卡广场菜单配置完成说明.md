# 知识库和人设卡广场菜单配置完成说明

## 概述

已完成知识库广场和人设卡广场页面的菜单和权限配置，用户可以通过左侧菜单访问这两个页面。

## 配置内容

### 1. 菜单配置 (init_menu.json)

在"内容管理"菜单下新增两个子菜单：

#### 知识库广场
- **名称**: 知识库广场
- **图标**: ele-Compass
- **路由**: /content/knowledge-square
- **组件**: content/knowledge/index
- **组件名**: contentKnowledgeSquare
- **排序**: 5
- **按钮权限**:
  - 查询 (knowledge_square:List)
  - 详情 (knowledge_square:Detail)
  - 收藏 (knowledge_square:Star)
  - 下载 (knowledge_square:Download)

#### 人设卡广场
- **名称**: 人设卡广场
- **图标**: ele-UserFilled
- **路由**: /content/persona-square
- **组件**: content/persona/index
- **组件名**: contentPersonaSquare
- **排序**: 6
- **按钮权限**:
  - 查询 (persona_square:List)
  - 详情 (persona_square:Detail)
  - 收藏 (persona_square:Star)
  - 下载 (persona_square:Download)

### 2. 角色菜单权限配置 (init_rolemenupermission.json)

为以下角色添加了菜单访问权限：

- **superadmin**: 可访问知识库广场和人设卡广场
- **admin**: 可访问知识库广场和人设卡广场
- **reviewer**: 可访问知识库广场和人设卡广场
- **user**: 可访问知识库广场和人设卡广场

### 3. 角色按钮权限配置 (init_rolemenubuttonpermission.json)

#### superadmin 和 admin
- 所有按钮权限，数据范围：全部数据 (data_range: 0)

#### reviewer
- 所有按钮权限，数据范围：全部数据 (data_range: 0)

#### user (普通用户)
- 查询、详情、下载：全部数据 (data_range: 0)
- 收藏：仅本人数据 (data_range: 4)

## 数据范围说明

- **0**: 全部数据
- **4**: 仅本人数据

普通用户可以查看和下载所有公开的知识库和人设卡，但只能管理自己的收藏。

## 初始化步骤

### 运行初始化命令

使用以下命令重新加载菜单和权限配置：

```bash
cd backend
conda activate mai_notebook
python manage.py init -app mainotebook.system -y
```

参数说明：
- `-app mainotebook.system`: 只初始化 system 应用
- `-y`: 重置模式，会覆盖现有数据

### 验证配置

登录系统后，检查：

1. 左侧菜单"内容管理"下是否显示"知识库广场"和"人设卡广场"
2. 点击菜单是否能正常跳转到对应页面
3. 页面功能是否正常（查询、详情、收藏、下载）
