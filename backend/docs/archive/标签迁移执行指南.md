## 常见问题

### Q1: 迁移后标签显示为空？

**A:** 检查数据是否为 `is_public=True`，标签统计只统计公开的记录。

```python
# 检查记录状态
PersonaCard.objects.filter(id='xxx').values('tags', 'is_public')
```

### Q2: 前端仍然发送字符串格式？

**A:** 检查前端 API 类型定义，确保 `tags` 字段类型为 `string[]`。

```typescript
// admin_web/src/api/persona.ts
export interface PersonaCardData {
    tags: string[]; // 应该是数组类型
}
```

### Q3: 标签统计数据不准确？

**A:** 重新运行 `rebuild_statistics()`：

```bash
python backend/manage.py shell -c "
from mainotebook.content.services.tag_service import TagService
result = TagService.rebuild_statistics()
print(result)
"
```

## 性能影响

- **数据库查询**: JSONField 查询性能与 TextField 相当
- **序列化开销**: 数组格式序列化更快（无需字符串分割）
- **前端处理**: 数组格式无需解析，性能更好
- **缓存策略**: 标签统计使用 Redis 缓存，1小时过期
