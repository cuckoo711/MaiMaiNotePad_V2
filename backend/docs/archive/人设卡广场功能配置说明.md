# 人设卡广场功能配置说明

## 完成时间

2026-03-03

## 功能概述

为人设卡实现了与知识库广场相同的功能，包括：
- 广场页面展示
- 标签筛选（人设卡专属标签）
- 收藏功能
- 热门推荐、最新上传、我的收藏分类

## 后端实现

### 1. 视图层修改

文件：`backend/mainotebook/content/views/persona_card.py`

在 `PersonaCardViewSet.get_queryset()` 方法中添加了收藏筛选支持：

```python
# 处理收藏筛选
starred_param = self.request.query_params.get('starred', '').lower()
if starred_param == 'true' and self.request.user.is_authenticated:
    # 只返回当前用户收藏的人设卡
    from mainotebook.content.models import StarRecord
    import uuid as uuid_module
    
    # 获取收藏的 target_id（字符串格式）
    starred_id_strs = StarRecord.objects.filter(
        user=self.request.user,
        target_type='persona'
    ).values_list('target_id', flat=True)
    
    # 将字符串转换为 UUID 对象
    starred_ids = []
    for id_str in starred_id_strs:
        try:
            starred_ids.append(uuid_module.UUID(id_str))
        except (ValueError, AttributeError):
            continue
    
    queryset = queryset.filter(id__in=starred_ids)
```

### 2. 标签统计

标签统计系统已支持人设卡类型（`tag_type='persona'`），通过以下 API 获取：

```
GET /api/content/tags/popular/?tag_type=persona&limit=12
```

### 3. API 端点

人设卡广场使用以下 API：

- `GET /api/content/persona/` - 获取人设卡列表
  - 支持参数：`page`, `page_size`, `search`, `ordering`, `starred`, `tags`
- `GET /api/content/persona/{id}/` - 获取人设卡详情
- `POST /api/content/persona/{id}/star/` - 收藏人设卡
- `DELETE /api/content/persona/{id}/unstar/` - 取消收藏
- `GET /api/content/persona/{id}/files/{file_id}/` - 下载文件
- `GET /api/content/tags/popular/?tag_type=persona` - 获取热门标签

## 前端实现

### 1. 文件结构

```
admin_web/src/views/content/persona/
├── api.ts          # API 接口定义
└── index.vue       # 人设卡广场页面
```

### 2. API 接口（api.ts）

提供以下接口函数：
- `getPersonaList(params)` - 获取人设卡列表
- `getPersonaDetail(id)` - 获取详情
- `starPersona(id)` - 收藏
- `unstarPersona(id)` - 取消收藏
- `downloadPersonaFile(id, fileId)` - 下载文件
- `getPopularTags(limit, tagType)` - 获取热门标签

### 3. 页面功能（index.vue）

**左侧导航：**
- 全部
- 热门推荐（按下载量排序）
- 最新上传（按创建时间排序）
- 我的收藏（只显示已收藏的）

**标签筛选：**
- 显示人设卡的热门标签
- 支持多标签筛选

**卡片展示：**
- 上传者头像
- 人设卡名称
- 描述（限制2行）
- 标签（最多显示3个，超出显示 +N）
- 下载次数和收藏数
- 收藏按钮

**详情弹窗：**
- 基本信息标签页
- 文件列表标签页
- 下载功能
- 收藏/取消收藏

## 与知识库广场的差异

| 功能 | 知识库广场 | 人设卡广场 |
|------|-----------|-----------|
| 标题 | 知识库广场 | 人设卡广场 |
| API 端点 | `/api/content/knowledge/` | `/api/content/persona/` |
| 标签类型 | `tag_type=knowledge` | `tag_type=persona` |
| 收藏类型 | `target_type=knowledge` | `target_type=persona` |
| 空状态提示 | 暂无知识库 | 暂无人设卡 |
| 详情标签 | 知识库名称 | 人设卡名称 |

## 注意事项

1. **标签隔离**：知识库和人设卡的标签是独立统计的，不会互相干扰
2. **收藏隔离**：收藏记录通过 `target_type` 字段区分类型
3. **权限控制**：确保用户有相应的权限才能访问人设卡广场
4. **UUID 转换**：收藏筛选时需要将字符串 ID 转换为 UUID 对象
