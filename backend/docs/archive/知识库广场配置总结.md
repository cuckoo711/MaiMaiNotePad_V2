## 技术要点

### 前端
- Vue 3 Composition API
- Element Plus UI 组件
- Pinia 状态管理
- 响应式布局
- 权限指令 `v-auth`

### 后端
- Django REST Framework
- ViewSet 架构
- 基于角色的权限控制 (RBAC)
- 数据范围过滤
- JSON Fixtures 初始化

### 权限控制
- 菜单级别：控制菜单显示
- 按钮级别：控制操作权限
- 数据级别：控制数据访问范围
  - 0: 全部数据
  - 4: 仅本人数据

## 注意事项

1. 前端组件路径必须与菜单配置中的 `component` 字段匹配
2. 组件名 `component_name` 必须唯一
3. 按钮权限的 `value` 必须与前端 `v-auth` 指令中使用的值一致
4. 数据范围 `data_range` 会影响后端 API 返回的数据
5. 初始化命令使用 `-y` 参数会覆盖现有数据，谨慎使用
